Що ж, почнемо з головної ідеї цієї місії. Тобі потрібно пройти по заданому рядку, взяти кожне слово і перевернути його. Наприклад, ти можеш використовувати цикл for або генератор списків (що є більш коротким і елегантним рішенням).

Як використати цикл for?

Давай я покажу тобі як працює цикл for. Наприклад, у нас є рядок "honey" і ми дуже хочемо по ньому пройтись циклом for і спробувати вивести кожну літеру. Тож код виглядатиме ось так: 

# Це початок циклу "for", де "і" це змінна       
# якій присвоюється кожна буква покроково
for i in "honey":  
    # Тут ми виведемо змінну "і"               
    print(i)

Вивід:

h
o
n
e
y

Тут як бачиш нічого складного. Ми отримаємо "h" на першому кроці, "o" на другому, «n» на третьому місці і так далі до кінця рядка.

Вау, добре! Тепер мені поясни, будь ласка, що таке генератор списків?

Генератор списків це легко! Генератор списку повертає список. За допомогою генераторів списку ми можемо записати вираз циклу for в один рядок! Наприклад зробимо так, щоб всі елементи в рядку "honey" стали подвійними. Ось код: 

a = "honey"
# Тут ми беремо кожну букву і подвоюємо її за допомогою множення на 2
b = [i*2 for i in a]
print(b)

Вивід:

['hh', 'oo', 'nn', 'ee', 'yy']

Як бачиш, у відповідь отримали список, але всі букви стали подвійними. Не переживай, якщо тобі потрібно все ж таки отримати рядок, то використай рядковий метод ' '.join(). 

# В дужки передаємо об'єкт, який хочемо "скласти" назад у рядок            
print(''.join(b))

Вивід:

'hhoonneeyy'

Або

# А якщо ми поставимо пробіл спочатку, то всі букви поєднає у рядок
# через пробіли. Детально читай за посиланням вище.      
print(' '.join(b))

Вивід:

'hh oo nn ee yy'

Як слова на букви розкласти мені ясно. Але як мені розкласти речення на слова?

Для такого випадку використовуй метод "str.split()". Це допоможе тобі розділити заданий рядок на список слів: 

# Після слова loves два пробіли            
d = "Bear loves  honey"  
c = d.split()
print(c)
# Але як бачиш пробіли зникли

Output:

['Bear', 'loves', 'honey']

Або

# Передамо в аргумент пробіл. Все на місці.            
c = d.split(' ')
print(c)

Вивід:

['Bear', 'loves', ' ', 'honey']

З циклами розібрались. А як мені перевернути слово?

Наприклад, у тебе є слово "honey" і тобі потрібно його перевернути. Звичайно, ти можеш скористатися методом reversed. Але ось більш короткий і простий спосіб: 

a = "honey"
print(a[::-1])

Вивід:

'yenoh'

Так, я маю усі елементи для успішного вирішення місії... Але я щось не можу закінчити її...

Твій остаточний код виглядатиме приблизно так. Тут нам знадобиться ще метод append() (почитай про нього). 

def backward_string_by_word(text: str) -> str:
    # Розділяємо на список
    new_text = text.split(' ')  
    result = []
    for i in new_text:
        # Перевертаємо і додаємо до списку
        result .append(i[::-1])
    # Об'єднуємо у рядок    
    return ' '.join(result)

Але цей код можна записати у зручному генераторі списків. Дивись: 

def backward_string_by_word(text: str) -> str:

    return ' '.join([x[::-1] for x in text.split(' ')])
