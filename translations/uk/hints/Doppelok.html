Я не знаю, як розпочати вирішення цієї місії
Що ж, почнемо з головної ідеї цієї місії. Тобі потрібно пройти по заданому рядку, взяти кожне слово і перевернути його. Наприклад, ти можеш використовувати "цикл for" або генератор списків (що є більш коротким і елегантним рішенням).
Отже, як мені перевернути слово?
Наприклад, у тебе є слово «мед», і тобі потрібно його перевернути. Звичайно, ти можеш скористатися інструкцією "reversed". Але ось більш короткий і простий спосіб: 
def a = "honey"
    a[::-1]  #>> 'yenoh'
Вау, добре! Тепер мені потрібна допомога з "циклом for" або генератором списків.
Генератор списків це легко! Отже, у випадку, коли нам потрібно створити новий список із рядка "Ведмеді люблять мед", ми можемо використати наступний вираз: 
a = "Bears loves honey"
b = [i for i in "Bears loves honey"]   # це генератор списків
# >> ['B', 'e', 'a', 'r', 's', ' ', 'l', 'o', 'v', 'e', 's', ' ', 'h', 'o', 'n', 'e', 'y']
Але мені потрібно ціле слово, а не букви.
Так, ти абсолютно правий! У цьому випадку використовуй метод "str.split()". Це допоможе тобі розділити заданий рядок на список слів: 
a = "Bears loves honey"
c = a.split()
# >> ['Bears', 'loves', 'honey']
А в іншому випадку ти можеш використовувати "' '.join()", щоб перетворити цей список на рядок назад. 
a = "Bears loves honey"
c = a.split()
# >> ['Bears', 'loves', 'honey']
d = " ".join(c)
# >> 'Bears loves honey'
Так, я маю усі елементи для успішного вирішення місії... Але я щось не можу закінчити її...
Твій остаточний код виглядатиме приблизно так: 
def backward_string_by_word(text: str) -> str:
    return ' '.join([x[::-1] for x in text.split(' ')])
