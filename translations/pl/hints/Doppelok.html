Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Cóż, proszę zacząć od głównej idei tej misji. Musi Pan przejść przez podany ciąg znaków, wziąć każde słowo i odwrócić je. Można na przykład użyć pętli for lub list comprehensions (co jest krótszym i bardziej eleganckim rozwiązaniem).

Jak używać pętli for?

Proszę pozwolić mi pokazać, jak działa pętla for. Na przykład, mamy ciąg "honey" i chcemy, aby przeszedł on przez pętlę for i spróbował wypisać każdą literę. Tak więc kod będzie wyglądał następująco: 

# This is the beginning of a "for" loop, where the "i"
# is the variable to which each letter is assigned one step at a time           
for i in "honey":  
    # Here we will output the variable "і"        
    print(i)

Wyjście:

h
o
n
e
y

Jak widać, nie ma tu nic skomplikowanego. Otrzymamy "h" w pierwszym kroku, "o" w drugim, "n" w trzecim i tak dalej, aż do końca ciągu.

Wow, dobrze! Teraz proszę mi wyjaśnić, co to jest list comprehensions?

Zrozumienie listy jest proste! Zrozumienie listy zwraca listę. Za pomocą list comprehensions możemy napisać wyrażenie pętli for w jednej linii! Na przykład, zróbmy tak, aby wszystkie elementy w wierszu "honey" stały się podwójne. Oto kod: 

a = "honey"
# Here we take each letter and double it by multiplying by 2
b = [i*2 for i in a]
print(b)

Wyjście:

['hh', 'oo', 'nn', 'ee', 'yy']

Jak widać, w odpowiedzi otrzymano listę, ale wszystkie litery stały się podwójne. Proszę się nie martwić, jeśli nadal chcesz uzyskać ciąg znaków, użyj metody string "''.join()". 

# In parentheses, we pass the object that we want to "fold"
# back into a string            
print(''.join(b))

Wynik:

'hhoonneeyy'

Lub 

# And if we put a space in '', then all letters will be combined into
# a string through spaces. Read in detail at the link above.
print(' '.join(b))

Wyjście:

'hh oo nn ee yy'

Jak podzielić słowa na litery jest dla mnie jasne. Ale jak podzielić zdanie na słowa?

W takim przypadku proszę skorzystać z metody str.split(). Pomoże ona Państwu podzielić podany ciąg znaków na listę słów: 

# Two spaces after the word loves            
d = "Bear loves  honey"
c = d.split()
# But as you can see, the spaces have disappeared
print(c)

Output:

['Bear', 'loves', 'honey']

Lub

# We pass a space in the argument. Everything is in place.
c = d.split(' ')
print(c)

Wyjście:

['Bear', 'loves', ' ', 'honey']

Jak więc odwrócić to słowo?

Na przykład, mają Państwo słowo "honey" i muszą je Państwo odwrócić. Oczywiście można użyć metody reversed. Ale tutaj jest krótsze i prostsze wyrażenie: 

a = "honey"
print(a[::-1])

Wyjście:

'yenoh'

Ok, mam wszystkie elementy do ukończenia tej misji... Ale czegoś mi brakuje...

Ostateczny kod będzie wyglądał mniej więcej tak. Tutaj będziemy również potrzebować metody .append(). 

def backward_string_by_word(text: str) -> str:
    # Divide into a list
    new_text = text.split(' ')  
    result = []
    for i in new_text:
        # Turn it over and add it to the list
        result .append(i[::-1])
    # Combine into a line     
    return ' '.join(result)

Ale ten kod można napisać w poręcznym generatorze list. Proszę spojrzeć: 

def backward_string_by_word(text: str) -> str:

    return ' '.join(x[::-1] for x in text.split(' '))
