Требуется обратить порядок букв в каждом слове предоставленной строки, так чтобы слова остались на своих местах.

Входные данные: строка.

Выходные данные: строка.

Примеры:

{% if interpreter.slug == "js-node" %} 
backwardStringByWord('') == ''
backwardStringByWord('world') == 'dlrow'
backwardStringByWord('hello world') == 'olleh dlrow'
backwardStringByWord('hello   world') == 'olleh   dlrow'
{% else %} 
backward_string_by_word('') == ''
backward_string_by_word('world') == 'dlrow'
backward_string_by_word('hello world') == 'olleh dlrow'
backward_string_by_word('hello   world') == 'olleh   dlrow'
{% endif %}

Предусловие: Строка состоит только из символов алфавита и пробелов.
