Comment utiliser une boucle for?

Laissez-moi vous montrer comment fonctionne la boucle for. Par exemple, nous avons la chaîne "honey" et nous voulons vraiment qu'elle passe par la boucle for et qu'elle essaie de sortir chaque lettre. Le code ressemblera donc à ceci: 

# This is the beginning of a "for" loop, where the "i"
# is the variable to which each letter is assigned one step at a time           
for i in "honey":  
    # Here we will output the variable "і"        
    print(i)

Sortie:

h
o
n
e
y

Comme vous pouvez le voir, il n'y a rien de compliqué ici. Nous obtiendrons "h" à la première étape, "o" à la deuxième, "n" à la troisième et ainsi de suite jusqu'à la fin de la chaîne.

Wow, c'est bien! Maintenant, expliquez-moi ce qu'est une liste de compréhension

La compréhension de liste, c'est facile! Une compréhension de liste renvoie une liste. Avec les compréhensions de liste, nous pouvons écrire l'expression de la boucle for en une seule ligne! Par exemple, faisons en sorte que tous les éléments de la ligne "honey" deviennent doubles. Voici le code: 

a = "honey"
# Here we take each letter and double it by multiplying by 2
b = [i*2 for i in a]
print(b)

Sortie:

['hh', 'oo', 'nn', 'ee', 'yy']

Comme vous pouvez le voir, il a reçu une liste en réponse, mais toutes les lettres sont devenues doubles. Ne vous inquiétez pas, si vous avez toujours besoin d'obtenir une chaîne de caractères, utilisez la méthode string "''.join()". 

# In parentheses, we pass the object that we want to "fold"
# back into a string            
print(''.join(b))

Résultat:

'hhoonneeyy'

Ou 

# And if we put a space in '', then all letters will be combined into
# a string through spaces. Read in detail at the link above.
print(' '.join(b))

Sortie:

'hh oo nn ee yy'

La manière de décomposer les mots en lettres est claire pour moi. Mais comment décomposer une phrase en mots?

Dans ce cas, utilisez la méthode str.split(). Elle vous aidera à diviser la chaîne donnée en une liste de mots: 

# Two spaces after the word loves            
d = "Bear loves  honey"
c = d.split()
# But as you can see, the spaces have disappeared
print(c)

Résultat:

['Bear', 'loves', 'honey']

Ou

# We pass a space in the argument. Everything is in place.
c = d.split(' ')
print(c)

Sortie:

['Bear', 'loves', ' ', 'honey']

Comment inverser le mot?

Par exemple, vous avez un mot "honey", et vous devez l'inverser. Bien sûr, vous pouvez utiliser la méthode reversed. Mais voici une expression plus courte et plus simple: 

a = "honey"
print(a[::-1])

Sortie:

'yenoh'

Votre code final ressemblera à ceci. Ici, nous aurons également besoin de la méthode .append(). 

def backward_string_by_word(text: str) -> str:
    # Divide into a list
    new_text = text.split(' ')  
    result = []
    for i in new_text:
        # Turn it over and add it to the list
        result .append(i[::-1])
    # Combine into a line     
    return ' '.join(result)

Mais ce code peut être écrit dans un générateur de liste pratique. Regardez: 

def backward_string_by_word(text: str) -> str:

    return ' '.join(x[::-1] for x in text.split(' '))
