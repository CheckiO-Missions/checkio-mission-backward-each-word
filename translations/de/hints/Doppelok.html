Wie verwendet man eine for Schleife?

Lassen Sie mich Ihnen zeigen, wie die for Schleife funktioniert. Wir haben zum Beispiel die Zeichenkette "honey" und möchten, dass sie die Schleife for durchläuft und versucht, jeden Buchstaben auszugeben. Der Code sieht also wie folgt aus: 

# This is the beginning of a "for" loop, where the "i"
# is the variable to which each letter is assigned one step at a time           
for i in "honey":  
    # Here we will output the variable "і"        
    print(i)

Ausgabe:

h
o
n
e
y

Wie Sie sehen können, ist hier nichts kompliziert. Im ersten Schritt erhalten wir "h", im zweiten Schritt "o", im dritten Schritt "n" und so weiter bis zum Ende der Zeichenkette.

Wow, gut! Erklären Sie mir jetzt bitte, was eine List Comprehensions ist?

Listenverstehen ist einfach! Ein Listenverständnis gibt eine Liste zurück. Mit List Comprehensions können wir den Ausdruck der Schleife for in eine Zeile schreiben! Lassen Sie uns zum Beispiel dafür sorgen, dass alle Elemente in der Zeile "honey" doppelt werden. Hier ist der Code: 

a = "honey"
# Here we take each letter and double it by multiplying by 2
b = [i*2 for i in a]
print(b)

Ausgabe:

['hh', 'oo', 'nn', 'ee', 'yy']

Wie Sie sehen, erhalten Sie als Antwort eine Liste, aber alle Buchstaben wurden verdoppelt. Keine Sorge, wenn Sie trotzdem eine Zeichenkette erhalten möchten, verwenden Sie die String-Methode "''.join()". 

# In parentheses, we pass the object that we want to "fold"
# back into a string            
print(''.join(b))

Ausgabe:

'hhoonneeyy'

Oder 

# And if we put a space in '', then all letters will be combined into
# a string through spaces. Read in detail at the link above.
print(' '.join(b))

Ausgabe:

'hh oo nn ee yy'

Wie man Wörter in Buchstaben zerlegt, ist mir klar. Aber wie zerlege ich einen Satz in Wörter?

Verwenden Sie für einen solchen Fall die Methode str.split(). Diese hilft Ihnen, die angegebene Zeichenkette in eine Liste von Wörtern zu zerlegen: 

# Two spaces after the word loves            
d = "Bear loves  honey"
c = d.split()
# But as you can see, the spaces have disappeared
print(c)

Ausgabe:

['Bear', 'loves', 'honey']

Oder

# We pass a space in the argument. Everything is in place.
c = d.split(' ')
print(c)

Ausgabe:

['Bear', 'loves', ' ', 'honey']

Wie kann ich das Wort umkehren?

Sie haben zum Beispiel ein Wort "honey" und müssen es umkehren. Natürlich können Sie die Methode reversed verwenden. Aber hier ist ein kurzer und einfacher Ausdruck: 

a = "honey"
print(a[::-1])

Ausgabe:

'yenoh'

Okay, ich habe alle Elemente, um diese Mission zu beenden... Aber mir fehlt noch etwas...

Ihr endgültiger Code wird in etwa so aussehen. Hier benötigen wir auch die Methode .append(). 

def backward_string_by_word(text: str) -> str:
    # Divide into a list
    new_text = text.split(' ')  
    result = []
    for i in new_text:
        # Turn it over and add it to the list
        result .append(i[::-1])
    # Combine into a line     
    return ' '.join(result)

Aber dieser Code kann in einem praktischen Listengenerator geschrieben werden. Sehen Sie: 

def backward_string_by_word(text: str) -> str:

    return ' '.join(x[::-1] for x in text.split(' '))
