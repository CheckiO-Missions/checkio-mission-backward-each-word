如何使用for 循环？

让我向您展示一下for 循环是如何工作的。例如，我们有一个字符串"honey" ，我们希望它通过for 循环并尝试输出每个字母。代码如下 

# This is the beginning of a "for" loop, where the "i"
# is the variable to which each letter is assigned one step at a time           
for i in "honey":  
    # Here we will output the variable "і"        
    print(i)

输出：

h
o
n
e
y

如您所见，这里并不复杂。第一步得到"h" ，第二步得到"o" ，第三步得到"n" ，以此类推直到字符串结束。

哇，很好！现在请向我解释一下什么是列表理解？

列表理解很简单！列表理解返回一个列表。使用列表理解，我们可以在一行中写出for 循环表达式！例如，让我们使"honey" 行中的所有元素都变成 double。下面是代码 

a = "honey"
# Here we take each letter and double it by multiplying by 2
b = [i*2 for i in a]
print(b)

输出：

['hh', 'oo', 'nn', 'ee', 'yy']

正如你所看到的，它收到了一个列表，但所有字母都变成了双倍。别担心，如果你仍然需要得到一个字符串，使用字符串方法"''.join()" 。 

# In parentheses, we pass the object that we want to "fold"
# back into a string            
print(''.join(b))

输出：

'hhoonneeyy'

或者 

# And if we put a space in '', then all letters will be combined into
# a string through spaces. Read in detail at the link above.
print(' '.join(b))

输出：或

'hh oo nn ee yy'

如何将单词分解成字母我很清楚。但如何将句子分解成单词呢？

在这种情况下，请使用str.split() 方法。这将帮助您将给定的字符串拆分成单词列表： 

# Two spaces after the word loves            
d = "Bear loves  honey"
c = d.split()
# But as you can see, the spaces have disappeared
print(c)

输出：

['Bear', 'loves', 'honey']

或

# We pass a space in the argument. Everything is in place.
c = d.split(' ')
print(c)

输出: Or

['Bear', 'loves', ' ', 'honey']

那么，如何反转单词呢？

例如，您有一个单词"honey" ，您需要将其反转。当然，您可以使用reversed 方法。但这里有更简短的表达方式： 

a = "honey"
print(a[::-1])

输出：

'yenoh'

您的最终代码将如下所示。这里我们还需要方法.append(). 

def backward_string_by_word(text: str) -> str:
    # Divide into a list
    new_text = text.split(' ')  
    result = []
    for i in new_text:
        # Turn it over and add it to the list
        result .append(i[::-1])
    # Combine into a line     
    return ' '.join(result)

但是这段代码可以写在一个方便的列表生成器中。请看 

def backward_string_by_word(text: str) -> str:

    return ' '.join(x[::-1] for x in text.split(' '))
