Well, let start with the main idea of this mission. You need to walk through the given string, take each word and reverse it. For example, you can use for loop or list comprehensions (which be more shortly and elegant solution).

How to use a for loop?

Let me show you how the for loop works. For example, we have the string "honey" and we really want it go through the for loop and try to output each letter. So the code will look like this: 

# This is the beginning of a "for" loop, where the "i"
# is the variable to which each letter is assigned one step at a time           
for i in "honey":  
    # Here we will output the variable "Ñ–"        
    print(i)

Output:

h
o
n
e
y

As you can see, there is nothing complicated here. We will get "h" in the first step, "o" at the second, "n" at the third and so on until the end of the string.

Wow, good! Now please explain to me what is a list comprehensions?

List comprehension is easy! A list comprehension returns a list. With list comprehensions we can write the for loop expression in one line! For example, let's make it so that all elements in the line "honey" became double. Here is the code: 

a = "honey"
# Here we take each letter and double it by multiplying by 2
b = [i*2 for i in a]
print(b)

Output:

['hh', 'oo', 'nn', 'ee', 'yy']

As you can see, it received a list in response, but all letters became double. Don't worry, if you still need to get a string, use the string method "''.join()". 

# In parentheses, we pass the object that we want to "fold"
# back into a string            
print(''.join(b))

Output:

'hhoonneeyy'

Or 

# And if we put a space in '', then all letters will be combined into
# a string through spaces. Read in detail at the link above.
print(' '.join(b))

Output:

'hh oo nn ee yy'

How to break words into letters is clear to me. But how do I break a sentence into words?

For such a case, use a method str.split(). This will help you to split the given string into a list of words: 

# Two spaces after the word loves            
d = "Bear loves  honey"
c = d.split()
# But as you can see, the spaces have disappeared
print(c)

Output:

['Bear', 'loves', 'honey']

Or

# We pass a space in the argument. Everything is in place.
c = d.split(' ')
print(c)

Output:

['Bear', 'loves', ' ', 'honey']

So, how do I reverse the word?

For example, you have a word "honey", and you need to reverse it. Of course, you can use reversed method. But here is more short and simple expression: 

a = "honey"
print(a[::-1])

Output:

'yenoh'

Your final code will look something like this. Here we will also need the method .append(). 

def backward_string_by_word(text: str) -> str:
    # Divide into a list
    new_text = text.split(' ')  
    result = []
    for i in new_text:
        # Turn it over and add it to the list
        result .append(i[::-1])
    # Combine into a line     
    return ' '.join(result)

But this code can be written in a handy list generator. Look: 

def backward_string_by_word(text: str) -> str:

    return ' '.join(x[::-1] for x in text.split(' '))
