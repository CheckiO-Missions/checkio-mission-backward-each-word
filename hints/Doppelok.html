Well, let start with the main idea of this mission. You need to walk through the given string, take each word and reverse it. For example, you can use "for" loop or list comprehensions (which be more shortly and elegant solution).
How to use a "for" loop?
Let me show you how the "for" loop works. For example, we have the string "honey" and we really want it go through the "for" loop and try to output each letter. So the code will look like this: 
for i in "honey":  # This is the beginning of a "for" loop, where the "i"
            # is the variable to which each letter is assigned one step at a time
    print(i)  # Here we will output the variable "Ñ–",
>> h  # We will get this in the first step
>> o  # We will get this in the second step
>> n  # We will get this in the third step and so on until the end of the string
>> e
>> y
As you can see, there is nothing complicated here.
Wow, good! Now please explain to me what is a list comprehensions?
List comprehensions is easy! A list comprehensions returns a list. With list comprehensions we can write the "for" loop expression in one line! For example, let's make it so that all elements in the line "honey" became double. Here is the code: 
a = "honey"
[i*2 for i in a]  # Here we take each letter and double it by multiplying by 2
>> ['hh', 'oo', 'nn', 'ee', 'yy']  # As you can see, it received a list in response
# but all letters became double
Don't worry, if you still need to get a string, use the string method "''.join()". 
a = 'honey'
b = [i*2 for i in a]
>> ['hh', 'oo', 'nn', 'ee', 'yy']
''.join(b)  # In parentheses, we pass the object that we want to "fold" back into a string
>> 'hhoonneeyy'
' '.join(b)  # And if we put a space in '', then all letters will be combined into a string
            # through spaces. Read in detail at the link above.
>> 'hh oo nn ee yy'

So, how do I reverse the word?
For example, you have a word "honey", and you need to reverse it. Of course, you can use instruction "reversed". But here is more short and simple expression: 
a = "honey"
a[::-1]  #>> 'yenoh'

Your final code will look something like this. Here we will also need the method ".append()". 
def backward_string_by_word(text: str) -> str:
    new_text = text.split(' ')  # Divide into a list
    result = []
    for i in new_text:
        result .append(i[::-1])  # Turn it over and add it to the list
    return ' '.join(result)  # Combine into a line
But this code can be written in a handy list generator. Look: 
def backward_string_by_word(text: str) -> str:
    return ' '.join([x[::-1] for x in text.split(' ')])
